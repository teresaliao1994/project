 一、

專案說明：
- 可以動態同步scss或sass在html上
- 此專案已引入bootstrap
- 將sass或scss檔放在src/css內，當編輯src/css檔內的sass和scss檔時會自動產生對應的css檔
------------------------------------------------------------------------------------
二、

使用說明：

第一次使用時，1~3都要做，
之後使用只要照3的方式做就好。

1. 全域安裝gulp
打開終端機 輸入 npm install gulp -g
-- 這條指令只要電腦有打過一次就不需要（所以你不需要打）
------------------------------------------
2. 安裝node_modules
-- 這條指令在新專案(沒有node_modules)的情況下才要打
（所以你新專案需要打）

步驟一：
打開終端機 並且來到這個資料夾的位置

example:
cd D:/project/gitandsassdemo
cd是進入的意思，空一格輸入這資料夾的位置

步驟二：

輸入npm install
意思就是在這個位置安裝node_modules
------------------------------------------
3. 在根目錄輸入gulp就可以使用專案

步驟一：
cd D:/project/gitandsassdemo

步驟二：
輸入gulp

------------------------------------------------------------------------------------
四、

路徑說明：

編輯src/css內的style.sass或src/js內的base.js練習
------------------------------------------------------------------------------------
五、

package.json紀錄了預設需要的程式工具
在打npm install的時候會依照這個檔案去產生node_modules

gulpfile.js就是我幫你寫的sass指令
打gulp的指令會去讀這個檔案，這個檔案會去幫你做sass轉譯

gulpfile.js要在滿足第二大項條件時，打gulp才會執行
